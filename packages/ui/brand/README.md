# Sendero Brand Assets

This directory keeps the design-system copy of Sendero's brand-book assets.

- `sendero_brand_book.pdf` is generated from the root `DESIGN.md` brand-book section.
- Storybook serves the same file from `apps/storybook/public/brand/sendero_brand_book.pdf`.
- Reusable brand tokens and usage guidance live in `packages/ui/src/brand/sendero-brand.ts`.

The primary mark is the vermillion binocular icon: discovery, wayfinding, and destination intelligence.

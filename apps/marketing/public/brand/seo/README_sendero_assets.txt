Sendero asset pack

This bundle includes:
- Transparent high-resolution PNG masters for the binocular icon in 4 approved colorways.
- Two master styles:
  1) textured = preserves the stamped editorial texture from the generated source artwork
  2) clean = flattened for better digital scaling in apps, favicons, and UI
- Platform/web icons:
  - app icon transparent 1024
  - store-ready app icon 1024 (solid cream background)
  - Android Chrome 512 and 192
  - Apple touch icon 180
  - X avatar 400
  - schema/logo assets 512 and 112
- Favicon bundle:
  - 16, 32, 48, 64 PNG
  - favicon.ico
- Social/SEO canvases:
  - Open Graph 1200x630
  - X card 1600x800
  - Google Discover 1600x900
  - X header 1500x500

Important note:
These files were derived from generated source artwork. They are suitable for presentation, MVP, web, and early product use, but a future vector redraw would improve edge quality, consistency, and scalability for very large print, signage, app-store polish, and ultra-small favicon rendering.

Transparency note:
The logo/icon masters and core digital icons are transparent PNGs.
The social/SEO canvases intentionally use solid backgrounds because social previews and search surfaces don't handle transparency consistently.

Sendero charges per tool call in USDC — no subscription, no seat license, no minimum. Pay for what your agent actually does.

## Tier comparison

| **Sandbox**   | Hackers, demos, prototyping            | Free (mock)    | 10,000 mock calls  | Community  |
| **Pay-go**    | Indie agents, side projects            | List price     | 0                  | Community  |
| **Studio**    | Startups shipping customer-facing bots | List price × 0.9 | $20 credit         | Slack      |
| **Partner**   | OTAs, TMCs, airlines                   | Negotiated      | Bespoke            | Dedicated  |

## Reading the meter

Every response carries a `meter` object. For on-chain tools it includes the Arc tx hash so you can reconcile against an independent block explorer:

```json
{
  "meter": {
    "tool": "settle_split",
    "usdc_atomic": "10000",
    "usdc_display": "0.01",
    "settled_tx": "0xabc..."
  }
}
```

## Failures are free

If a tool errors before producing a result, no payment is captured. The EIP-3009 authorization you signed is discarded server-side; no refund is needed.

## Ethereum mainnet comparison

For reference, a single USDC transfer on Ethereum mainnet (30 gwei, ETH at $3,400) costs roughly **$2.14**. A Sendero on-chain tool call at $0.003 is **~700× cheaper** — and carries the Duffel call, the policy check, and the meter receipt at the same time.

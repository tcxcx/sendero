Sendero is a production travel API wearing two faces.

- **For humans**, it's a chat that books flights and settles the four-way commission fan-out the moment the ticket issues.
- **For agents**, it's 14 MCP tools exposed over x402 — every call paid for in USDC with no invoicing, no keys, no retries.

## What you can build

- A concierge that searches flights, quotes FX, and books all within one conversation.
- A backoffice that reconciles commissions across an airline, an agency, a TMC, and a traveler in a single atomic settle.
- An agent-to-agent marketplace where another AI pays Sendero per tool call and walks away with real PNRs.

## Built on

- [Circle Arc](https://www.circle.com/arc) — USDC as native gas, sub-second finality.
- [Duffel](https://duffel.com) — real flight inventory and PNR issuance.
- [x402](https://x402.org) — HTTP-native agentic payments.
- Model Context Protocol — tool-aware LLM integration.
